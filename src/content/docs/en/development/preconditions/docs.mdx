# IDE: IntelliJ
## Plugin: ktlint
- Install https://plugins.jetbrains.com/plugin/15057-ktlint
- Update Settings -> Tools -> Actions on Save: 
  - Enable: `Format with ktlint`
- Update Settings -> Tools -> KtLint
  - Enable: `Mode` -> `Distract free (recommended)`
  - Enable: `on save`

# Agentic: Claude Code
## Install helper libraries *in GitBash*
- Clone `app-server` git repository: https://github.com/dfp-ch/app-server
- Open GitBash terminal in the `app-server`'s root directory (**ADMIN MODE**)
- Run the following commands to install dependencies:
```./scripts/install-libraries```
- Note: `uninstall-libraries` does uninstall the libraries again

# GIT
## Install pre-commit hooks *in GitBash*
- Clone `app-server` git repository: https://github.com/dfp-ch/app-server
- Open GitBash terminal in the `app-server`'s root directory (**ADMIN MODE**)
- Run the following commands to install dependencies:
```./scripts/install-git-hooks```
- Note: `uninstall-git-hooks` does uninstall the hooks again
- Note: ktlint plugin of IntelliJ IDEA will automatically format even before pre-commit hooks are executed
